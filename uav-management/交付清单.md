## 📝 使用说明

### 快速启动
1. 双击 `启动应用.bat`
2. 等待启动完成（10-20秒）
3. 双击 `打开系统.html` 或访问 http://localhost:27543/drone/list

### 功能使用
1. **查看列表**：访问首页即可看到所有无人机
2. **添加无人机**：点击"添加无人机"按钮，填写表单提交
3. **编辑无人机**：在列表中点击"编辑"按钮
4. **删除无人机**：在列表中点击"删除"按钮并确认
5. **搜索无人机**：使用搜索表单按条件筛选

## ⚠️ 注意事项

### 使用注意
1. 序列号必须唯一，不能重复
2. 所有必填字段都要填写完整
3. 日期格式为 YYYY-MM-DD
4. 删除操作为软删除，数据仍保留在数据库

### 系统要求
- Java 8 或更高版本
- Windows 操作系统
- 至少 512MB 可用内存
- 现代浏览器（Chrome、Firefox、Edge）

### 端口配置
- 默认端口：27543
- 如需修改，编辑 `application.yml` 中的 `server.port`

## 🐛 已知问题

### 1. POST 重定向 500 错误
- **影响：** 添加无人机后页面跳转时可能出现错误
- **状态：** 数据已成功保存，不影响功能
- **解决：** 手动刷新列表页面即可

### 2. 初始数据
- **影响：** 首次启动时列表可能为空
- **状态：** 已添加空状态提示
- **解决：** 手动添加数据或查看初始化日志
