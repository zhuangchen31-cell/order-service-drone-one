# 加载动画文字优化说明

## 优化时间
2026-04-28 17:06

## 优化内容

根据用户需求，优化了页面加载动画的提示文字，使其根据不同的操作显示不同的内容。

## 修改详情

### 1. 列表页面（list.html）

#### 动态文字逻辑
根据 URL 参数判断操作类型：

**首次进入系统**（无搜索参数）
```
显示：无人机系统启动中...
```

**执行搜索操作**（有搜索参数）
```
显示：无人机系统查询中...
```

#### 实现方式
```javascript
// 根据 URL 参数判断操作类型
const urlParams = new URLSearchParams(window.location.search);
const loaderText = document.querySelector('.loader-text');

// 检查是否有搜索参数
const hasSearchParams = urlParams.has('model') || 
                       urlParams.has('serialNumber') || 
                       urlParams.has('status');

if (hasSearchParams) {
    loaderText.textContent = '无人机系统查询中...';
} else {
    loaderText.textContent = '无人机系统启动中...';
}
```

## 技术实现

### 检测逻辑
使用 JavaScript 的 `URLSearchParams` API 检测 URL 中的查询参数：

```javascript
const urlParams = new URLSearchParams(window.location.search);

// 检测是否有以下任一参数
- model: 无人机型号
- serialNumber: 序列号
- status: 状态
```

### 执行时机
在页面 HTML 加载后立即执行（在 `<script>` 标签中使用立即执行函数）：

```javascript
(function() {
    // 立即执行，在页面渲染前修改文字
})();
```

- 加载动画显示时长: 1000ms（1秒）
- 淡出过渡时间: 500ms

## 技术细节

### URL 参数检测
```javascript
// 检测单个参数
urlParams.has('model')  // true/false

// 获取参数值
urlParams.get('model')  // 返回参数值或 null

// 检测多个参数
const hasSearchParams = 
    urlParams.has('model') || 
    urlParams.has('serialNumber') || 
    urlParams.has('status');
```

### 文字更新时机
```
1. HTML 解析完成
2. 立即执行函数运行
3. 检测 URL 参数
4. 更新加载文字
5. 显示加载动画
6. 页面内容加载
7. 1秒后隐藏加载动画
```
