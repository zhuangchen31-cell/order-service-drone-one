# 无人机管理系统 - 使用说明

## 快速开始

### 方法一：使用启动脚本（推荐）
1. 双击 `启动应用.bat` 文件
2. 等待应用启动完成（看到 "Started UavManagementApplication" 提示）
3. 打开浏览器访问：http://localhost:27543/

### 方法二：使用命令行
```bash
cd uav-management
apache-maven-3.8.8\bin\mvn.cmd spring-boot:run
```

## 功能说明

### 1. 查看无人机列表
- 访问：http://localhost:27543/drone/list
- 功能：
  - 查看所有无人机信息
  - 按型号、制造商、状态搜索
  - 分页浏览
  - 编辑、删除、生成AI属性

### 2. 添加无人机
- 访问：http://localhost:27543/drone/add
- 必填信息：
  - 无人机型号
  - 序列号（必须唯一）
  - 制造商
  - 购买日期
  - 状态

### 3. 编辑无人机
- 在列表页面点击"编辑"按钮
- 修改信息后提交

### 4. 删除无人机
- 在列表页面点击"删除"按钮
- 确认后删除（软删除，数据仍保留在数据库）

## 常见问题

### Q: 提示"序列号已存在"怎么办？
A: 每架无人机的序列号必须唯一，请更换一个新的序列号。

### Q: 添加成功后看不到数据？
A: 请刷新列表页面，或检查搜索条件是否过滤了新添加的数据。

### Q: 如何停止应用？
A: 在命令行窗口按 `Ctrl+C` 停止应用。

### Q: 数据保存在哪里？
A: 数据保存在 `data/uav.db` SQLite 数据库文件中。

### Q: 如何重置数据库？
A: 删除 `data/uav.db` 文件，重新启动应用即可自动创建新数据库。

- Java 8 或更高版本
- Windows 操作系统
- 至少 512MB 可用内存

## 端口配置
- 默认端口：27543
- 如需修改，请编辑 `src/main/resources/application.yml` 文件中的 `server.port` 配置
